# On-Demand-Exam
📚 On-Demand Exam Management System

A Django-based web application that allows institutions to manage and conduct on-demand examinations efficiently.
The system supports student registration, exam scheduling, room assignment, attendance management, and result management through an admin-controlled platform.

🚀 Features

👨‍🎓 Student Registration & Login

✅ Admin Approval for Students

📝 Exam Scheduling

📅 On-Demand Exam Requests

🏫 Room Assignment System

📊 Attendance Sheet Management

🧾 Result Management by Admin

📚 Subject & Semester Management

🔐 Role-based Authentication

🛠️ Tech Stack
Technology	  Usage
Python	    Backend programming
Django	    Web framework
SQLite	    Default database
HTML/CSS	  Frontend
Bootstrap	  UI styling
